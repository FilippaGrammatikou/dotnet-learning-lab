# LINQ_TicketProgram

> C# console project focused on querying and shaping in-memory ticket data with LINQ.

This project uses a small ticket-based dataset to practice working with object collections in C#, with emphasis on filtering, ordering, projection, grouping, and safe lookups.

## What it demonstrates

- custom object modelling with `Ticket`
- enum-based status values with `TicketStatus`
- filtering tickets by status
- sorting tickets by creation date
- projecting data into simplified summary shapes
- grouping and counting by status
- recent-date filtering
- safe lookup logic with `FirstOrDefault`

## Context
This project sits between basic C# syntax practice and more backend-oriented thinking, especially around querying, shaping, and validating data from collections.
